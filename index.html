About Me
I am a musician/web developer from the US.

I started in 2017, making simple remixes of songs like Toby Fox - Megalovania. Eventually I got a Midi Keyboard and started making my own compositions.
At that point, I decided to unlist all of my previous songs because well, they sucked.

In early 2017, I also started making web apps like discord bots.
Ayumi, my first discord bot, was coded in JavaScript (NodeJS) , originally using discord.io , but was later re-written to use discord.js

Currently I run a discord bot named Sami. More information about Sami can be found here!
